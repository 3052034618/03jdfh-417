# 测试场景1: 循环积累诅咒等级

* [恐惧值] 玩家的恐惧等级，越高越容易触发特殊事件

## 第一章: 测试循环

### start
@开始
@节奏: 平静
@标题: 起始房间

房间里有一面镜子和一扇门。
镜子似乎可以反复查看，每次看都会增加恐惧值。

- [a1] 查看镜子 -> look_mirror
- [a2] 走向大门 -> door_hall

### look_mirror
@节奏: 异常
@标题: 看向镜子

你看向镜子，镜中的你露出诡异的微笑。

> ++恐惧值

- [b1] 继续看镜子 -> look_mirror
- [b2] 离开镜子 -> start

### door_hall
@节奏: 平静
@标题: 门廊

你来到大门前。根据恐惧值的不同，会发生不同的事情。

- [c1]? 恐惧值>=3 推开血门 -> high_fear_ending
- [c2]? 恐惧值<3 推开普通门 -> normal_ending
- [c3]? !恐惧值 轻轻开门 -> peaceful_ending

### normal_ending
@结局
@节奏: 平静
@标题: 普通结局

你推开大门，阳光洒进来。你平安离开了这座房子。

### peaceful_ending
@结局
@节奏: 平静
@标题: 平静结局

你甚至没有感受到任何恐惧，就轻松离开了。

### high_fear_ending
@结局
@节奏: 崩坏
@标题: 恐惧结局

? 恐惧值>=3

镜子里的你走了出来，代替了你。你永远被困在了镜中世界。
