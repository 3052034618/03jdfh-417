# 镜中 Manor - 恐怖剧本示例

* [镜中影] 镜中的倒影开始有了自己的意志，等级越高越频繁干涉现实
* [门后低语] 走廊尽头传来的呓语，等级越高越容易让人失去理智
* [旧伤复发] 主角童年的创伤被唤起，影响判断力
* [遗忘之雾] 玩家开始遗忘重要信息，剧情分支发生变化

## 第一章: 抵达

### ch1_start
@开始
@节奏: 平静
@标题: 雾中抵达

汽车引擎在浓雾中熄火，你站在一座废弃庄园的铁门前。
雨丝打湿了你的外套，门内隐约透出微弱的光亮。

- [a1] 推开铁门进入 -> ch1_hall
- [a2] 绕到房子后面看看 -> ch1_backyard
- [a3] 返回车内等待雾散 -> bad_end_coward

### ch1_hall
@节奏: 平静
@标题: 大厅

大厅里布满灰尘，水晶吊灯已经破碎。
墙上挂满了老旧的肖像画，画中人的眼睛似乎在跟随你移动。
壁炉上方有一面巨大的镀金镜框，但镜子被白布遮盖着。

> +镜中影

- [b1] 揭开镜子上的白布 -> ch1_mirror_reveal
- [b2] 检查壁炉 -> ch1_fireplace
- [b3] 走进东侧走廊 -> ch2_corridor_start

### ch1_backyard
@节奏: 异常
@标题: 后院

后院杂草丛生，一口被封死的古井引起了你的注意。
井口用粗大的铁链缠绕，上面贴满了褪色的符咒。

- [c1] 尝试打开井盖 -> ch1_well_open
- ![c2] 对着井口喊话 -> ch1_well_yell
- [c3] 返回前门 -> ch1_hall

### ch1_mirror_reveal
@节奏: 异常
@标题: 镜中世界

白布落下，你看到镜中的自己...但有些不对劲。
镜中人的嘴角挂着你没有的微笑。

> ++镜中影

- [d1] 打碎镜子 -> ch1_mirror_break
- [d2] 仔细观察镜中细节 -> ch1_mirror_observe
- [d3] 移开目光转身离开 -> ch2_corridor_start

### ch1_fireplace
@节奏: 平静
@标题: 壁炉密道

壁炉里似乎有东西在发光。
你伸手进去，发现了一条隐藏的通道。

> +门后低语

-> ch2_secret_passage

### ch1_well_open
@节奏: 追逐
@标题: 井底之物

铁链应声断裂，井盖被你掀开。
一股腐臭气息涌出，紧接着，一只苍白的手从井里伸了出来！

> ++门后低语
> +旧伤复发

- [e1] 拼命逃跑 -> ch1_hall
- [e2] 被拖入井中 -> bad_end_well

### ch1_well_yell
@节奏: 异常
@标题: 回声

"有人吗？"
你的声音在井中回荡，然后...有人回应了。
那是你自己的声音，但充满了痛苦和绝望。

"救救我...为什么要把我留在这里..."

> +镜中影

- [f1] 伸手去拉 -> bad_end_well
- [f2] 恐惧后退 -> ch1_backyard

### ch1_mirror_break
@节奏: 追逐
@标题: 碎裂诅咒

镜子碎裂的瞬间，无数碎片朝你飞来！
你护住头部，但还是被划伤了手臂。
当你再次睁开眼，发现周围的一切都变了...

> ++镜中影
> +遗忘之雾

-> ch2_corridor_distorted

### ch1_mirror_observe
@节奏: 揭露
@标题: 镜中真相

你凑近镜子，看到镜中的背景里站着一个人影。
那个人影...和你长得一模一样，但浑身是血。
镜子上用血迹写着："不要相信你看到的"

> +镜中影

-> ch2_corridor_start

## 第二章: 走廊

### ch2_corridor_start
@节奏: 平静
@标题: 无尽走廊

走廊两侧是一扇扇紧闭的房门。
地毯吸走了你的脚步声，四周静得可怕。
你数着门牌号：101、102、103...103？刚才不是已经数过103了吗？

- [g1] 随便选一扇门进去 -> ch2_room_random
- [g2] 寻找楼梯 -> ch2_stairs
- [g3] 检查墙上的画 -> ch2_paintings

### ch2_corridor_distorted
@节奏: 异常
@标题: 扭曲回廊

走廊在你眼前扭曲，墙壁开始融化。
门牌数字在不断变化，时不时出现"404"或"666"。
你听到身后传来沉重的呼吸声。

? 镜中影>=2
> ++遗忘之雾

- [h1] 疯狂向前跑 -> ch2_stairs_distorted
- [h2] 转身面对呼吸声 -> ch2_face_breath
- [h3] 随便推开一扇门 -> ch2_room_nightmare

### ch2_secret_passage
@节奏: 异常
@标题: 密道深处

密道狭窄而潮湿，墙壁上爬满了不知名的黑色藤蔓。
前方传来断断续续的低语声，似乎在念着你的名字。

> ++门后低语

- [i1] 继续前进 -> ch2_secret_room
- [i2] 原路返回 -> ch1_hall

### ch2_room_random
@节奏: 平静
@标题: 随机房间

你随意推开一扇门，里面是一间普通的卧室。
床上躺着一个玩偶，它的头转向了你。

> +门后低语

- [j1] 拿起玩偶检查 -> ch2_doll_check
- [j2] 快速离开房间 -> ch2_corridor_start

### ch2_stairs
@节奏: 平静
@标题: 楼梯间

楼梯间的窗户被木板封死，只有微弱的光线从缝隙中透入。
向下的楼梯被坍塌的 debris 堵住，只有向上的路还通着。

-> ch3_floor2_start

### ch2_paintings
@节奏: 异常
@标题: 肖像画廊

墙上的肖像画记录着这座庄园历任主人的面貌。
你注意到越近期的画像，人物的面部越模糊。
最后一幅画是空白的，但画框上刻着今天的日期。

> +镜中影

- [k1] 刮掉最后一幅画的表面 -> ch2_painting_scratch
- [k2] 返回走廊 -> ch2_corridor_start

### ch2_stairs_distorted
@节奏: 追逐
@标题: 狂奔

你拼命向前跑，走廊似乎没有尽头。
身后的呼吸声越来越近，你能感觉到它冰冷的气息拂过你的后颈。

> ++门后低语
> ++旧伤复发

-> ch3_floor2_start

### ch2_face_breath
@节奏: 揭露
@标题: 直面恐惧

你猛地转身，看到了...你自己。
那个"你"浑身是血，眼睛是完全的黑色。
"终于找到你了，"它说，"现在，把身体还给我。"

? 镜中影>=3
> ++镜中影

- [l1] 攻击它 -> bad_end_mirror_twin
- [l2] 接受它 -> good_end_accept
- [l3] 逃跑 -> ch3_floor2_start

### ch2_room_nightmare
@节奏: 崩坏
@标题: 噩梦房间

房间里是你童年的卧室。
你的父母躺在床上，一动不动。
墙上用血迹写着："这都是你的错"

? 旧伤复发
> ++旧伤复发
> ++门后低语

-> bad_end_nightmare

### ch2_secret_room
@节奏: 揭露
@标题: 密室

密道尽头是一间小密室。
墙上贴满了关于你的新闻剪报，从你出生到今天，应有尽有。
桌上有一本日记，最后一页写着："祭品已就位，仪式即将开始。"

> +镜中影
> +门后低语

- [m1] 带走日记 -> ch3_floor2_start
- [m2] 继续搜索密室 -> ch2_secret_search

### ch2_doll_check
@节奏: 异常
@标题: 诅咒玩偶

玩偶的肚子被缝合过，里面似乎有东西。
你拆开缝线，掉出一张照片——是你小时候的照片。
背面写着："献给最爱的孩子"

> +旧伤复发
> -镜中影

-> ch2_corridor_start

### ch2_painting_scratch
@节奏: 揭露
@标题: 画中秘密

你刮掉颜料，下面露出一幅完整的肖像。
画中的人...就是你。
但这幅画至少有一百年的历史了。

> ++镜中影
> +遗忘之雾

-> ch3_floor2_start

### ch2_secret_search
@节奏: 异常
@标题: 更多发现

你在密室角落发现了一个铁箱。
里面装着一套沾满血迹的仪式服，还有一本古老的咒语书。

> +镜中影

-> ch3_floor2_start

## 第三章: 结局

### ch3_floor2_start
@节奏: 异常
@标题: 二楼大厅

二楼的大厅中央是一个巨大的五芒星图案。
五芒星的五个角各放着一支蜡烛，火焰是诡异的蓝色。
大厅尽头的大门上刻着你的名字。

- [n1] 走进大门 -> ch3_final_room
- [n2] 检查五芒星 -> ch3_pentagram
- [n3] 尝试从窗户逃跑 -> ch3_window_escape

### ch3_final_room
@节奏: 揭露
@标题: 真相之室

房间里站着一个穿着黑袍的人。
他缓缓摘下面具...是你的父亲。
"你终于来了，"他说，"我等这一天等了二十年。"

> ++镜中影
> ++门后低语

- [o1] 听他解释 -> good_end_truth
- [o2] 攻击他 -> bad_end_patricide
- [o3] 转身逃跑 -> neutral_end_escape

### ch3_pentagram
@节奏: 崩坏
@标题: 仪式启动

你的脚刚碰到五芒星，蜡烛火焰突然暴涨！
五芒星开始发光，你感觉到有什么东西正在进入你的身体。
镜中影、门后低语、所有的诅咒...它们正在融合。

? 镜中影>=2
? 门后低语>=2
> ++镜中影
> ++门后低语
> ++旧伤复发

- [p1] 抵抗融合 -> neutral_end_resist
- [p2] 接受融合 -> true_end_godhood

### ch3_window_escape
@节奏: 追逐
@标题: 破窗

你冲向窗户，用椅子砸碎玻璃。
外面的雾更浓了，但你能听到远处传来的警笛声。
你跳了下去...

> -镜中影
> -门后低语

-> neutral_end_escape

### bad_end_coward
@结局
@节奏: 平静
@标题: 懦夫结局

你回到车里，锁上车门，试图忽略那座庄园。
但当你发动汽车时，后视镜里出现了一个人影。
"胆小鬼，"镜中的你说，"你以为你逃得掉吗？"
汽车失控冲出公路，最后人们只找到了一辆空车。

? !镜中影

### bad_end_well
@结局
@节奏: 崩坏
@标题: 井底亡魂

冰冷的手抓住了你的脚踝，你被拖入黑暗的井底。
井水灌入你的肺，在意识消散的最后一刻，
你看到水面倒映出无数张脸——
它们都和你长得一模一样。

### bad_end_mirror_twin
@结局
@节奏: 崩坏
@标题: 镜中替身

你的攻击穿过了它的身体，就像打在水面上。
它笑着握住你的手腕，你们的皮肤开始融合。
"谢谢你的身体，"它说，"现在，我来替你活下去。"
三天后，"你"回到了家，但没有人发现你已经不是你了。

### bad_end_nightmare
@结局
@节奏: 崩坏
@标题: 永恒噩梦

你尖叫着醒来，发现自己又回到了这个房间。
一次又一次，你不断经历着同样的噩梦。
这是诅咒，也是惩罚。
你将永远困在自己的记忆中，一遍遍重温那最痛苦的一天。

### bad_end_patricide
@结局
@节奏: 崩坏
@标题: 弑父者

你冲上去攻击他，但他只是轻轻挥了挥手。
你的身体僵住了，无法动弹。
"愚蠢的孩子，"他叹息道，"你以为我没有准备吗？"
你成为了仪式的祭品，你的灵魂被永远囚禁在镜中世界。

### good_end_accept
@结局
@节奏: 揭露
@标题: 接受自我

你不再抵抗，任由它靠近。
当你们的身体融合时，你终于明白了一切。
它是你的一部分，是你多年来压抑的负面情绪。
接受它，你获得了完整。
你带着全新的力量走出庄园，浓雾已经散去。

### good_end_truth
@结局
@节奏: 揭露
@标题: 真相大白

你的父亲告诉你一切——
你天生拥有特殊的能力，而这座庄园是一个封印。
那些"诅咒"其实是在保护你，防止你的力量失控。
"现在你长大了，"他说，"是时候继承这一切了。"
你成为了庄园的新主人，守护着这个世界不被黑暗侵蚀。

### neutral_end_escape
@结局
@节奏: 平静
@标题: 逃离

你拼命奔跑，终于冲出了庄园的大门。
警笛声越来越近，你安全了。
但每当你照镜子时，总能看到一个模糊的人影在身后。
你逃离了庄园，但你知道，它还在等着你回去。

? !镜中影
? !门后低语

### neutral_end_resist
@结局
@节奏: 异常
@标题: 抵抗者

你用尽全力抵抗融合，诅咒开始反噬。
你的一只眼睛变成了纯黑色，那是诅咒留下的印记。
仪式失败了，但诅咒永远不会离开你。
你带着这只"诅咒之眼"活了下去，能看到别人看不到的东西。
这是恩赐，还是诅咒？也许两者都是。

### true_end_godhood
@结局
@节奏: 崩坏
@标题: 成神之路

你张开双臂，任由所有诅咒融入你的身体。
痛苦过后，是前所未有的力量。
你不再是人类，也不再是鬼魂。
你成为了新的神，这座庄园只是你无尽领域的一部分。
无数个世界在你眼前展开，而你，将是它们永恒的主宰。

? 镜中影>=3
? 门后低语>=3
? 旧伤复发>=2
