# 核心场景测试剧本

## 第一章: 入口

### start
@开始
@节奏: 平静
@标题: 神社门口

你站在神社门口。
旁边有一条通往森林的小路，听说森林里闹鬼。

- [a1] 直接进去 -> shrine_yard
- [a2] 先去森林看看 -> forest_path

### forest_path
@节奏: 异常
@标题: 森林小径

森林里阴气很重。
你越往里走，越觉得压抑。

> +怨念

- [b1] 继续深入 -> forest_deep
- [b2] 返回神社 -> start
- [b3] 在周围找找 -> forest_search

### forest_search
@节奏: 异常
@标题: 森林搜索

你在森林里找到了一个护身符。
但你的不安感更强了。

> +护符
> ++怨念

- [c1] 返回神社 -> shrine_yard
- [c2] 再找找 -> forest_search_2

### forest_search_2
@节奏: 追逐
@标题: 继续搜索

你又找到了一些东西，但那个东西...追上来了！

> ++怨念

- [d1] 跑回神社 -> shrine_yard
- [d2] 我要回家！ -> bad_end_coward

### forest_deep
@节奏: 追逐
@标题: 森林深处

森林深处怨气冲天。
你看到一个黑影向你飘来...

> +++怨念

? 护符
@标题: 护符发光

危急时刻，护身符发出了光芒！
黑影尖叫着退散了。

> --护符
> -怨念

-> shrine_yard

? 怨念>=4
@标题: 被抓住

你被黑影抓住了！
意识开始模糊...

-> bad_end_possessed

## 第二章: 神社

### shrine_yard
@节奏: 平静
@标题: 神社庭院

神社庭院很安静。
正殿的门开着，里面传来诵经声。

> +怨念

- [e1] 进正殿 -> shrine_hall
- [e2] 去侧殿 -> shrine_side
- [e3] 去后院 -> shrine_backyard
- [e4] 回森林 -> forest_path
- [e5] 出去喘口气 -> outside_rest

### outside_rest
@节奏: 平静
@标题: 外面休息

你跑到神社外面，深吸了几口气。
一离开神社范围，压抑感就减轻了。

> --怨念

- [f1] 再进去 -> shrine_yard
- [f2] 算了，回家吧 -> bad_end_coward

### shrine_hall
@节奏: 平静
@标题: 正殿

正殿里很庄严。
一位神官正在念经。
"施主，要净化一下吗？"

- [g1] 接受净化 -> purification
- [g2] 婉拒，四处看看 -> shrine_hall_look
- [g3] 离开 -> shrine_yard

### purification
@节奏: 平静
@标题: 净化仪式

神官为你举行了净化仪式。
你感到身心都清净了。

> -怨念

- [h1] 谢谢神官 -> shrine_yard_pure

### shrine_hall_look
@节奏: 异常
@标题: 正殿探索

你在正殿里四处张望。
墙上的壁画描绘着古代的驱魔仪式。

> +怨念

- [i1] 离开 -> shrine_yard

### shrine_side
@节奏: 异常
@标题: 侧殿

侧殿里放着很多古老的法器。
你好像听到了什么声音...

> ++怨念

- [j1] 进去看看 -> side_room
- [j2] 离开 -> shrine_yard

### side_room
@节奏: 追逐
@标题: 侧殿深处

它朝你扑了过来！

> ++怨念

? 护符
@标题: 护符退敌

> --护符
> -怨念

-> shrine_yard_pure

? 怨念>=3
@标题: 被怨灵追上

-> bad_end_ghost

### shrine_backyard

@标题: 后院

- [k2] 欣赏樱花 -> cherry_tree
- [k3] 离开 -> shrine_yard

### old_well

## 第三章: 结局

### bad_end_coward
@结局
@节奏: 平静
@标题: 胆小鬼结局

你决定不冒险了，转身就走。
走出没多远，你听到身后传来沉重的门铃声。
你不敢回头，一路小跑离开了。
也许，你错过了什么，但至少你安全了。

? 怨念<2

### bad_end_possessed
@结局
@节奏: 崩坏
@标题: 附身结局

你被怨灵附身了。

? 怨念>=4
? !护符

### bad_end_ghost
@结局
@节奏: 崩坏

? 怨念>=3
? !护符

### bad_end_well

@节奏: 崩坏
