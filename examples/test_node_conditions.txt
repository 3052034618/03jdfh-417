# 测试场景2: 节点进入条件控制与未定义诅咒

* [已定义诅咒] 这个诅咒在开头有定义

## 第一章: 条件测试

### start
@开始
@节奏: 平静
@标题: 开始

- [a1] 走左边 -> left_room
- [a2] 走右边 -> right_room_entrance
- [a3] 直接离开 -> exit_early

### left_room
@节奏: 平静
@标题: 左边房间

左边房间有一个宝箱，打开后获得未定义的诅咒。

> +未定义诅咒A
> +未定义诅咒B

- [b1] 回去 -> start
- [b2] 去右边 -> right_room_entrance

### right_room_entrance
@节奏: 平静
@标题: 右边房间门口

右边有两个房间，只有被诅咒的人才能看到隐藏的那个。

- [c1] 进入普通房间 -> right_room_normal
- [c2]? 未定义诅咒A 进入隐藏房间 -> right_room_secret

### right_room_normal
@节奏: 平静
@标题: 普通右边房间

右边房间什么也没有，空荡荡的。

- [d1] 离开 -> start

### right_room_secret
@节奏: 异常
@标题: 隐藏房间

你看到了只有被诅咒者才能看到的景象。
墙上用鲜血写着古老的文字。

> +已定义诅咒

- [e1] 深入调查 -> secret_ending
- [e2] 逃离 -> start

### exit_early
@结局
@节奏: 平静
@标题: 提前离开结局

你没有深入探索，直接离开了这座房子。

### secret_ending
@结局
@节奏: 揭露
@标题: 秘密结局

? 已定义诅咒
? 未定义诅咒A

你发现了这座房子的真正秘密——它是一座诅咒的祭坛，而你就是下一个祭品。
但同时你也获得了操控诅咒的力量。
