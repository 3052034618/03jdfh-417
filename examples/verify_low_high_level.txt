# 验收剧本: 先低等级开门再绕回刷高等级
# 关键验证点：
#   1. 通风口(怨念<=2)在低等级时可进入 → 不应被报为死选项
#   2. 绕回刷高等级后，深处走廊也可进入
#   3. 同一节点同时存在低等级和高等级入口状态
#   4. 回地面净化后可再进入，保持清净状态
#   5. boss_room 使用选项条件而非节点条件来避免互斥条件问题

* [怨念] 怨灵气息

## 第一章: 入口

### start
@开始
@节奏: 平静
@标题: 地下入口

你发现了一个地下遗迹的入口。
空气中弥漫着若有若无的怨气。

- [a1] 直接下去 -> underground_hall
- [a2] 先在周围找找 -> surface_search

### surface_search
@节奏: 异常
@标题: 地面搜索

你在周围发现了一些痕迹。
不小心沾上了一些怨气。

> +怨念

- [b1] 下去 -> underground_hall
- [b2] 再找找 -> surface_search_2

### surface_search_2
@节奏: 追逐
@标题: 继续搜索

你找到了一面破旧的护符。
但怨气更重了...

> ++怨念

- [c1] 赶紧下去 -> underground_hall
- [c2] 再找找 -> surface_loop

### surface_loop
@节奏: 追逐
@标题: 反复搜索

你又找到了一些东西。
但那个东西越来越近了...

> ++怨念

- [d1] 下去 -> underground_hall
- [d2] 还想找 -> surface_loop

## 第二章: 地下

### underground_hall
@节奏: 平静
@标题: 地下大厅

地下大厅很宽阔。
有几条路可以走。

> +怨念

- [e1] 走通风口 -> ventilation
- [e2] 走主通道 -> main_corridor
- [e3] 回地面 -> surface_return

### surface_return
@节奏: 平静
@标题: 回到地面

你回到地面，呼吸了几口新鲜空气。
怨气减轻了。

> --怨念

- [f1] 再下去 -> underground_hall
- [f2] 继续搜索 -> surface_search

### ventilation
@节奏: 平静
@标题: 通风口

? 怨念<=2
@标题: 狭窄通风口

通风口很窄，只有怨气不重的人才能挤过去。
你勉强挤了进去。

- [g1] 往前爬 -> vent_passage
- [g2] 退回去 -> underground_hall

### vent_passage
@节奏: 平静
@标题: 通风管道

通风管道通向一个隐藏的房间。

- [h1] 进入隐藏房间 -> hidden_room
- [h2] 爬回去 -> underground_hall

### hidden_room
@节奏: 揭露
@标题: 隐藏房间

你来到了一间隐藏的房间。
这里竟然完全没有怨气！
房间中央放着一颗发光的宝珠。

- [i1] 拿起宝珠 -> treasure_take
- [i2] 离开 -> underground_hall

### treasure_take
@节奏: 揭露
@标题: 拿起宝珠

你拿起了宝珠。
宝珠发出温暖的光芒，驱散了你身上的怨气。

> -怨念

-> good_ending_treasure

### main_corridor
@节奏: 异常
@标题: 主通道

主通道很长，怨气越来越重。

> ++怨念

- [j1] 继续走 -> deep_corridor
- [j2] 回大厅 -> underground_hall

### deep_corridor
@节奏: 追逐
@标题: 深处走廊

走廊深处怨气冲天。
你感觉有什么东西在前方等着你...

> ++怨念

- [k1] 跑！ -> underground_hall
- [k2]?怨念>=5 正面迎战 -> bad_end_devoured
- [k3]?怨念>=3 正面迎战 -> normal_ending_battle
- [k4]?怨念<3 正面迎战 -> normal_ending_easy

### bad_end_devoured
@结局
@节奏: 崩坏
@标题: 吞噬结局

怨灵太强大了！
你被完全吞噬...
从此以后，地下遗迹又多了一个新的怨灵。

? 怨念>=5

### normal_ending_battle
@结局
@节奏: 追逐
@标题: 苨战结局

你拼尽全力战斗！
虽然伤痕累累，但你活了下来。

? 怨念>=3
? 怨念<5

### normal_ending_easy
@结局
@节奏: 平静
@标题: 轻松结局

你轻松击败了弱小的怨灵。
这没什么难的。

? 怨念<=2

### good_ending_treasure
@结局
@节奏: 揭露
@标题: 宝藏结局

你带着宝珠离开了地下遗迹。
宝珠的光芒驱散了所有的怨气。
你成为了新一代的守护者。

? !怨念
