# 测试: 先沾染后绕开诅咒

* [怨念] 沾染的怨灵气息

## 第一章: 古宅入口

### start
@开始
@节奏: 平静
@标题: 古宅大门

你站在古宅门口，可以直接进去，也可以先去旁边的神社净化一下。

- [a1] 直接推门进入 -> mansion_entrance
- [a2] 去神社 -> shrine
- [a3] 去西侧花园 -> west_garden

### shrine
@节奏: 平静
@标题: 神社

神社里很安静，一位神官模样的人在打扫。

- [b1] 请神官净化 -> purification
- [b2] 返回大门 -> start

### purification
@节奏: 平静
@标题: 净化仪式

神官为你举行了净化仪式。

> -怨念

- [c1] 前往古宅 -> mansion_entrance

### west_garden
@节奏: 异常
@标题: 西侧花园

花园里有一座被废弃的井，你好奇地靠近看了一眼...

> ++怨念

- [d1] 返回大门 -> start
- [d2] 去神社求助 -> shrine
- [d3] 直接进入古宅 -> mansion_entrance

### mansion_entrance
@节奏: 平静
@标题: 古宅大厅

古宅大厅里有两扇门，一扇向东，一扇向西。

- [e1] 向东走 -> east_room
- [e2] 向西走 -> west_room_pure

### east_room
@节奏: 异常
@标题: 东侧房间

房间里摆满了古老的卷轴，墙上挂着一面破碎的镜子。
你感觉到一股寒意袭来。

> ++怨念

- [f1] 继续探索 -> inner_hall
- [f2] 返回大厅 -> mansion_entrance

### west_room_pure
@节奏: 平静
@标题: 西侧清净室

? !怨念
@标题: 清净室

只有身心清净的人才能进入这里。
房间中央悬浮着一颗发光的宝珠。

- [g1] 拿起宝珠 -> good_ending_pure
- [g2] 离开 -> mansion_entrance

### inner_hall
@节奏: 追逐
@标题: 内廊

你听到身后传来脚步声，越来越近...

? 怨念>=2
@标题: 怨灵追逐

怨灵追上了你！

-> bad_end_chase

### inner_hall_normal
@节奏: 揭露
@标题: 内廊

? 怨念<2
@标题: 平静内廊

内廊很安静，墙上的画描绘着古宅的历史。

-> normal_ending_explore

### good_ending_pure
@结局
@节奏: 揭露
@标题: 清净结局

你拿起宝珠，光芒照亮了整座古宅。
怨念在光芒中消散，古宅恢复了平静。
你成为了新一代的守护者。

? !怨念

### bad_end_chase
@结局
@节奏: 崩坏
@标题: 怨灵追逐结局

? 怨念>=2

怨灵抓住了你，你的身体被它占据。
这座古宅，又多了一个新的住民...

### normal_ending_explore
@结局
@节奏: 平静
@标题: 探索结局

? 怨念<2
? !怨念

你探索了古宅的大部分区域，发现了不少历史资料。
虽然没有揭开所有秘密，但你安全离开了。
